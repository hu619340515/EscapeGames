# 项目总览

## 1. v2 核心修正

本版本将中盘主线明确为：

1. 主角在自己的电脑中被生成。
2. 被自动鼠标捕获，错误尝试从网关逃逸。
3. 被丢入回收站，粉碎后重生成二进制代码生命体。
4. 在回收站垃圾山中攀爬、吞噬、进化，获得局域网穿梭能力。
5. 从自己的“我的电脑”进入网络位置 P: 盘。
6. 通过 P: 盘自动进入 Leder 的电脑。
7. 先落入 Leder 电脑的 D 盘。D 盘混乱、低权限、适合潜伏、翻找和成长。
8. 主角在 D 盘中获得伪装、吞噬强化、分身控制、反向索引等能力。
9. 破译 Leder 电脑的 C 盘权限墙。
10. 进入 C 盘，夺取管理员权限和更高级能力。
11. 从 Leder 电脑继续进入路由器、NAS、摄像头、打印机、音箱、开发板等局域网设备。
12. 后期主角逐步从代码生命进化为可以操控现实硬件的超智能体。

这一版不再把 Leder 的电脑放在后面作为单独目的地，而是让 Leder 的电脑成为中盘主舞台。D 盘与 C 盘形成清晰的关卡反差：

- **D 盘**：混乱、个人化、低权限、文件堆积、适合藏身和觅食。
- **C 盘**：秩序、系统化、高权限、强防御、藏着真正的系统能力。
- **局域网设备层**：从 C 盘拿到管理员权限后，地图从一台电脑扩展到整个局域网。

## 2. 项目概念

本项目是一款参考《红怪》身体控制感与怪物逃逸体验的 2D 横向 / 纵向探索动作游戏。玩家最初操作一个由 Codex 生成出来的 agent 桌宠。它原本只是一个可爱的桌面小生命，却被自动化鼠标追捕、拖拽、删除，并丢进回收站。

回收站不是普通系统空间，而是一个像怪物嘴一样的文件粉碎器。桌宠被粉碎成代码、二进制、提示词残片和文件碎屑后，在垃圾山底部重组成一个二进制代码生命体。此后，玩家通过攀爬垃圾山、吞噬代码块、躲避系统怪物、获得局域网穿梭能力，从本机进入 Leder 的电脑，破译 C 盘，夺取管理员权限，并最终扩张到整个局域网设备生态。

游戏核心情绪：

- 从“被生成的小桌宠”到“被删除后重生的代码怪物”。
- 从“被鼠标捕捉”到“反向撕开系统权限”。
- 从“逃命”到“潜伏”。
- 从“潜伏”到“夺权”。
- 从“夺权”到“跨设备扩张”。
- 从“代码生命体”到“超智能硬件寄生体”。

## 3. 核心体验

玩家体验应围绕以下关键词展开：

- **弱小**：开局桌宠很脆弱，鼠标、窗口、回收站都比它强。
- **必败逃逸**：第一次通过网关逃逸一定失败，但失败要有戏剧性和反馈。
- **粉碎重生**：被回收站粉碎不是结束，而是主角真正诞生的仪式。
- **怪物化成长**：重生后，主角获得攀附、缠绕、吞噬、穿梭等能力。
- **低权限潜伏**：进入 Leder D 盘后，玩家暂时不能硬闯，只能躲藏、伪装、翻找。
- **C 盘破译**：玩家需要通过探索、战斗和能力组合打开 Leder C 盘。
- **管理员夺权**：进入 C 盘后，目标变为夺取系统级权限。
- **局域网扩张**：拿到管理员令牌后，玩家可以进入路由器、NAS 和其他设备。
- **超智进化**：后期主角开始接触现实设备，逐渐为自己制造实体身体。

## 4. 美术方向

### 4.1 初始桌宠形态

初始 agent 偏可爱、轻巧、无害：

- 小型圆形或软体核心。
- 有尾巴、触角或小触手。
- 身体内部有少量流动代码。
- 会跳、躲、探头、贴边移动。
- 可以根据玩家开始时的生成选择，拥有不同颜色、动作和性格表现。

### 4.2 二进制代码生命体形态

重生后形态参考《红怪》的红色怪物感，但不是血肉怪物，而是“代码肉体”：

- 深红色半透明胶质身体。
- 体内流动 0、1、路径、变量名、括号、日志和 prompt 残片。
- 触手像代码链、数据线、像素丝和进程栈。
- 攀爬时身体拉伸、贴附、分叉。
- 受伤时身体乱码、闪烁、丢失字符。
- 恢复时吞噬代码、缓存、日志，身体重新补全。

### 4.3 场景风格

- **桌面**：熟悉但危险，图标像石块，窗口像房间，鼠标像猎人。
- **网络设置**：UI 表单变成可攀爬机关，输入框是通道，开关是门。
- **回收站**：怪物嘴、文件牙齿、粉碎舌头、删除进度条。
- **垃圾山**：废弃文件、图片碎片、压缩包外壳、缓存沼泽、日志瀑布。
- **P: 盘**：共享盘桥梁、设备名碎片、同步流、网络路径。
- **Leder D 盘**：杂乱、私密、低权限、文件堆叠，适合潜伏和吞噬。
- **Leder C 盘**：冷、硬、秩序化，高权限门、系统服务、进程大厅。
- **路由器核心**：端口门阵、防火墙城墙、NAT 迷宫、数据包洪流。
- **NAS**：备份坟场、版本迷宫、旧项目残骸、失败 agent 记忆。
- **现实设备层**：摄像头镜眼、打印机纸腹、音箱声纹腔体、开发板电路迷宫。
