# Agent 桌宠逃逸游戏方案索引

这个目录把原来的单文件方案拆成了“总览、章节、全局设定”三类。原始总稿保留为：[agent桌宠逃逸游戏设计方案v2.md](agent桌宠逃逸游戏设计方案v2.md)。

## 建议使用方式

- 写具体关卡、演出、地图、剧情时，只改对应的 章节/第XX章_*.md。
- 改角色能力、战斗、成长数值时，优先改 [全局设定/核心玩法系统.md](全局设定/核心玩法系统.md)。
- 改敌人或 Boss 时，优先改 [全局设定/敌人与Boss.md](全局设定/敌人与Boss.md)。
- 改分支、结局、真相线时，优先改 [全局设定/后续故事线.md](全局设定/后续故事线.md) 和 [全局设定/关键剧情节点与结局.md](全局设定/关键剧情节点与结局.md)。
- 新章节细化时，可以参考 [章节/_章节开发模板.md](章节/_章节开发模板.md)。
