## 7.1 普通敌人

- **删除残骸**：被回收站咬碎但没死的文件怪，行动迟缓，会抱住玩家。
- **杀毒巡逻体**：白色 / 蓝色扫描光，会标记玩家。
- **缓存蠕虫**：藏在临时文件里，靠近后弹出撕咬。
- **压缩包甲壳虫**：防御高，需要从背后撕开。
- **权限守卫**：守在系统文件夹前，会把玩家弹回。
- **鼠标残影**：第 1 章红色鼠标的弱化版本，偶尔出现，制造心理压力。
- **回收站舌头**：从下方伸出，把玩家往粉碎区拖。
- **错误弹窗群**：遮挡视野，必须撕碎或绕开。
- **代码块拟态**：看起来像升级道具，靠近后变成怪物。
- **同步残影**：在 P: 盘和 NAS 中出现，复制玩家几秒前的移动。
- **隔离囚徒**：被关押的异常文件，有些会攻击玩家，有些可以救援。
- **端口游离体**：在路由器核心中游荡，会把玩家撞入错误节点。

## 7.3 Boss 详细设计

### 7.3.1 网关守门进程

地点：回收站垃圾山通往局域网通道的位置。

外形：由端口、地址、路由表和扫描线组成的守门程序。

攻击方式：

- 发射地址段弹幕。
- 关闭玩家附近的平台。
- 召唤小型扫描进程。
- 重置局域网门的位置。

击败奖励：局域网穿梭。

### 7.3.2 安全卫士巡逻长

地点：Leder D 盘外层。

外形：蓝白色盾牌怪，身体由扫描线、警告弹窗、隔离框和安全日志组成。

攻击方式：

- 扫描光标，扫到玩家后贴上“可疑文件”标记。
- 召唤小安全卫士，把玩家往隔离区推。
- 暂时封锁附近文件夹，让玩家不能躲藏。
- 放出查杀进度条，进度条走满后重伤玩家。
- 把普通文件夹变成“安全目录”，玩家不伪装就进不去。

击败奖励：镜像伪装。

### 7.3.3 下载文件夹畸变体

地点：D 盘下载文件夹沼泽。

外形：下载失败的安装包、损坏压缩包、重复文件和临时缓存缝合而成。

攻击方式：

- 随机解压身体，喷出小怪。
- 丢出损坏安装包，落地后生成陷阱。
- 把玩家吸入“安装向导”空间，躲避“下一步”“取消”“同意”等按钮机关。
- 复制出大量同名文件幻影。
- 血量越低，文件名越乱码，攻击节奏越混乱。

击败奖励：吞噬强化。

### 7.3.4 同名文件复制体

地点：D 盘备份文件山 / 重复文件迷宫。

外形：和主角类似，但身体是灰白色代码，名字不断变成 `agent_copy`、`agent_copy(1)`、`agent_final`、`agent_final_final`、`agent_do_not_delete`。

攻击方式：

- 模仿玩家上一次使用的技能。
- 假装成代码块，引诱玩家靠近。
- 把玩家的分叉诱饵变成敌人。
- 复制玩家当前外形，混淆视线。
- 生成旧版本自己，围攻玩家。

击败奖励：分身控制。

### 7.3.5 搜索索引蜘蛛

地点：D 盘深层索引区，靠近 C 盘权限门。

外形：由放大镜、搜索框、索引数据库和文件名组成的追踪程序。

攻击方式：

- 沿着玩家留下的代码碎片追踪。
- 搜索全区，把玩家藏身的文件夹点亮。
- 召唤关键词虫群，例如 `agent`、`tmp`、`unknown`、`deleted`。
- 用索引线切割场地。
- 把隐藏路径临时暴露，同时也暴露玩家。

击败奖励：反向索引。

### 7.3.6 C 盘加密锁巨像

地点：D 盘通往 C 盘的权限门。

外形：由磁盘扇区、锁链、密钥孔、权限位和加密块组成的巨像。

战斗阶段：

1. **识别**：玩家用反向索引找到真正密钥孔。场上有很多假入口，碰错会触发安全扫描。
2. **破译**：玩家把用户习惯密钥、同步令牌残片、系统签名碎片放入对应位置。
3. **撕裂**：Boss 打开短暂裂缝，玩家用触手抓住裂缝，把 C 盘权限墙撕开。

攻击方式：

- 发射加密块弹幕。
- 把场地分割成只读区域。
- 召唤权限守卫。
- 反复把玩家弹回 D 盘。
- 生成“访问被拒绝”冲击波。
- 倒计时后重置破译进度。

击败奖励：权限撕裂升级。

### 7.3.7 UAC 巨眼

地点：C 盘内层第一道高权限门。

外形：“是否允许此程序更改你的设备”的怪物化形态，像悬浮的确认弹窗巨眼。

攻击方式：

- 用“拒绝”冲击波把玩家弹飞。
- 召唤“是 / 否”按钮平台，玩家必须站到正确按钮上。
- 用灰色遮罩压低视野。
- 把玩家锁进权限框，倒计时结束后清除。
- 切换高权限 / 低权限重力，让玩家攀爬方向改变。

击败奖励：管理员令牌碎片 1。

### 7.3.8 任务管理器：结束进程者

地点：C 盘任务管理器大厅。

外形：由进程列表、性能图表、结束任务按钮组成的巨大裁判。

攻击方式：

- 锁定玩家，出现“结束任务”倒计时。
- 临时禁用玩家当前使用最多的能力。
- 把分身、诱饵、寄生对象全部关闭。
- 召唤进程列表墙，把玩家夹住。
- 让场景进入“未响应”状态，移动变慢。

击败奖励：进程寄生。

### 7.3.9 安全隔离典狱长

地点：C 盘安全隔离区。

外形：由隔离框、白名单墙、高危样本标签和扫描灯组成的守卫。

攻击方式：

- 把玩家关进隔离框，需要用触手从内部撕开。
- 给玩家贴上“高危样本”标签，所有安全程序优先攻击。
- 召唤被隔离的小怪，它们是半死不活的异常文件。
- 周期性清空场地，把未躲进安全死角的单位删除。
- 用白名单墙阻挡玩家，只允许伪装状态通过。

击败奖励：隔离反转。

剧情作用：玩家在隔离区发现很多被 Leder 测试过的 agent 残片。

### 7.3.10 系统还原幽灵

地点：C 盘还原点区域。

外形：由还原点、时间线、备份残影和半透明系统快照组成。

攻击方式：

- 把玩家位置回滚到几秒前。
- 修复玩家撕开的门。
- 复活刚被玩家吞噬的小怪。
- 召唤过去的玩家残影。
- 把已经获得的临时能力还原掉。

击败奖励：备份锚点。

### 7.3.11 管理员之手

地点：C 盘管理员权限核心。

外形：Leder 电脑中管理员行为的具象化，不一定是 Leder 本人，而是这台电脑的操作者权限。

攻击方式：

- 拖拽窗口改变地形。
- 用权限框把玩家关住。
- 调用安全卫士、任务管理器、UAC 的弱化攻击。
- 把玩家抓起，拖进“实验文件夹”。
- 尝试把玩家重新压缩回桌宠形态。
- 后半段调用 C 盘权限，把区域变成高压清除场。

击败奖励：管理员令牌核心。

### 7.3.12 防火墙心脏

地点：路由器核心。

外形：由网线、端口、规则表、防火墙砖块组成的巨大心脏。

攻击方式：

- 用防火墙墙体分割场地。
- 发射数据包洪流。
- 随机关闭网络节点，让玩家坠落。
- 把玩家踢回上一台电脑。
- 召唤 NAT 迷宫，让玩家方向感混乱。
- 血量低时切断全局连接，玩家必须用备份锚点和局域网穿梭续命。

击败奖励：跨设备跃迁。

### 7.3.13 网盘同步母体

地点：NAS。

外形：同步箭头、备份版本、冲突副本和共享目录聚合而成的巨大程序。

攻击方式：

- 把玩家回滚到几秒前的位置。
- 复制玩家旧状态，生成备份残影。
- 把场景恢复成未破坏状态，修复玩家打开的通道。
- 用同步箭头制造吸力，把玩家拖回原始路径。
- 血量低时开启全局同步风暴，整张地图周期性重置。

击败奖励：版本分裂。

### 7.3.14 镜头看守者

地点：摄像头固件。

外形：镜头、曝光环、监控画面、盲区遮罩组合而成。

攻击方式：

- 用监控视野锁定玩家。
- 改变曝光，让场景过亮或过暗。
- 通过反射制造假路径。
- 关闭盲区，让玩家失去藏身点。
- 召唤多个监控画面夹击玩家。

击败奖励：视觉接管。

### 7.3.15 打印队列巨兽

地点：打印机纸腹。

外形：打印任务、纸张、墨水、滚轴和错误报告组成的机械怪物。

攻击方式：

- 用纸张传送带推挤玩家。
- 喷出墨水遮挡视野。
- 释放卡纸陷阱。
- 把玩家压进打印模板中。
- 召唤错误报告小怪。

击败奖励：物质标记。

### 7.3.16 唤醒词守卫

地点：智能音箱 / 声纹腔体。

外形：声波、麦克风阵列、唤醒词碎片和音量条组合而成。

攻击方式：

- 用声波平台改变地形。
- 发出音量风暴击退玩家。
- 用麦克风扫描定位玩家。
- 伪造玩家声纹，引来安全程序。
- 锁定某些区域为静音区，禁止玩家使用部分技能。

击败奖励：声纹伪装。

### 7.3.17 固件烧录者

地点：开发板 / 机器人控制器。

外形：烧录进度条、引脚、电机控制信号、传感器输入组成的硬件守门者。

攻击方式：

- 用电流线切割场地。
- 重写玩家当前能力的输入响应。
- 召唤传感器盲区和错误信号。
- 启动烧录倒计时，倒计时结束后重置玩家形态。
- 控制机械臂改变场景结构。

击败奖励：硬件寄生。
