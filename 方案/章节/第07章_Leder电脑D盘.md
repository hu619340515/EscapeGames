# 第 7 章：Leder 电脑 D 盘

[返回方案索引](../README.md) | [核心玩法系统](../全局设定/核心玩法系统.md) | [敌人与 Boss](../全局设定/敌人与Boss.md)

## 章节开发定位

- **定位**：中盘主舞台上半场：低权限潜伏、翻找、成长。
- **主要结果**：获得镜像伪装、吞噬强化、分身控制、反向索引，并集齐 C 盘钥匙碎片。
- **关联 Boss / 威胁**：安全卫士巡逻长、下载文件夹畸变体、同名文件复制体、搜索索引蜘蛛

## 正文

D 盘是玩家进入 Leder 电脑后的第一个主关卡。这里非常乱，是最适合代码生命潜伏的地方。

D 盘关卡关键词：

- 低权限。
- 文件混乱。
- 可藏身点多。
- 敌人巡逻密集但不够聪明。
- 有大量可吞噬资源。
- 藏着破译 C 盘所需线索。

主要区域：

- **下载文件夹沼泽**：安装包、压缩包、临时文件、重复文件堆在一起。
- **项目文件夹森林**：Leder 的开发项目、测试文件、代码片段堆成树状结构。
- **素材库洞穴**：图片、音频、视频、贴图、模型文件组成复杂地形。
- **备份文件山**：旧版本文件堆叠，适合放过去版本谜题。
- **重复文件迷宫**：大量同名文件和副本，引出复制体 Boss。
- **临时同步区**：P: 盘同步到 D 盘的节点，是返回局域网的重要位置。
- **C 盘权限门外层**：D 盘深处通往 C 盘的高墙，玩家一开始无法进入。

玩家目标：

1. 躲避安全卫士巡逻。
2. 收集 Leder 的使用痕迹。
3. 找到 C 盘加密 / 权限门的线索。
4. 获得镜像伪装。
5. 获得吞噬强化。
6. 获得分身控制。
7. 获得反向索引。
8. 收集破译 C 盘所需的三个钥匙碎片。
