# 第 9 章：Leder 电脑 C 盘

[返回方案索引](../README.md) | [核心玩法系统](../全局设定/核心玩法系统.md) | [敌人与 Boss](../全局设定/敌人与Boss.md)

## 章节开发定位

- **定位**：中盘主舞台下半场：高权限夺权。
- **主要结果**：获得进程寄生、隔离反转、备份锚点、管理员令牌核心，打开局域网设备层。
- **关联 Boss / 威胁**：UAC 巨眼、任务管理器、安全隔离典狱长、系统还原幽灵、管理员之手

## 正文

C 盘和 D 盘形成强烈反差。D 盘乱，C 盘冷。D 盘像文件堆，C 盘像系统城堡。

C 盘关卡关键词：

- 高权限。
- 系统服务。
- 进程寄生。
- 隔离区。
- 还原点。
- 管理员令牌。

主要区域：

- **Program Files 外墙**：软件安装目录变成高墙，每个程序都有自己的防御机制。
- **AppData 暗巷**：缓存、配置、会话、用户数据藏在这里，适合潜行。
- **启动项走廊**：敌人刷新点，玩家学习进程寄生。
- **系统服务管道**：后台服务像管道运行，玩家可以附着穿过安全门。
- **安全隔离区**：被杀毒软件隔离的文件关在这里，藏有其他 agent 碎片。
- **System32 远景区**：前期只给玩家看，不完全开放，制造深层权限诱惑。
- **任务管理器大厅**：所有运行中程序像笼子一样排列，是中段 Boss 区。
- **管理员权限核心**：Leder 电脑篇大 Boss 区域。

玩家目标：

1. 击败 UAC 巨眼，拿到管理员令牌碎片。
2. 击败任务管理器，获得进程寄生。
3. 进入安全隔离区，发现 agent 被封存的线索。
4. 击败系统还原幽灵，获得备份锚点。
5. 击败管理员之手，夺取管理员令牌核心。
6. 打开局域网设备层。
